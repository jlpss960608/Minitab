# Minitab Application

## Overview
This application provides statistical analysis and data manipulation functionalities.

## Features
- Data Handling: Read and clean data from CSV, XLSX, or Numbers files.
- Statistical Analysis: Perform various statistical tests including T-tests, Mann-Whitney U, and Chi-Square.
- User Interface: macOS-optimized Tkinter widgets.
- Graphics Generation: Summary statistics visualizations including histograms and probability plots.

## Installation
Use the following command to install the dependencies:

```
pip install -r requirements.txt
```
